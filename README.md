
# Space Cat Games - The free and open source way to host a games website!
Ever wanted to run a games website, but don't know how to code? Space Cat Games solves this problem! It is small and easy to deploy, just unzip the source code into the root of your website and boom! It's there!

## Contributors and Automations

### meowcat767-auto
Meowcat767-auto's job is to run chore tasks. It's runner is located at `worker.py`.
